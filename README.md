# presence-tool
A prototype of the new presence tool for Chamilo

## Project setup
```
yarn install
```

### Compiles and runs the server
```
yarn express:run
```

### Compiles and hot-reloads the UI for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
